## Installation

```bash
poetry install --with dev
```

## Usage

If you're running the app locally, please add a `.env` file at the root of the project with your crendentials:

```bash
VANNA_API_KEY=...
GCP_PROJECT_ID=...
```

Then run the app with this command:

```bash
poetry run streamlit run app.py
```
